## 📌 Project Overview

Customer Transaction Prediction is a **Machine Learning classification project** that predicts whether a customer will perform a transaction or not based on anonymized numerical features.

The goal of this project is to analyze customer data and build a predictive model that can help organizations understand **customer transaction behavior**.

---

## 🎯 Problem Statement

Financial institutions and businesses need to identify whether a customer is likely to make a transaction.

Using **machine learning algorithms**, we analyze historical customer data and build models that can predict customer transaction behavior.

**Target Variable**

| Value | Meaning |
|------|--------|
| 0 | Customer did NOT make a transaction |
| 1 | Customer made a transaction |
